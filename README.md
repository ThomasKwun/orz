# ORZ
## DESC
该项目是个人学习练手使用，主要探索研究VUE3+TS的使用，还有研究微前端Qiankun解决方案。

因为是微前端方案，ORZ是主应用，功能模块会拆分到子应用，主应用只包含登录、框架页，通用组件等。

ORZ-DEV-SERVER，是一个接口代理工具，允许代理到本地静态的json文件，如果本地没有这个接口文件，允许代理到服务器。


## 技术选型

`Vue3`
`Typescript`
`Qiankun`
`Vue-cli`
